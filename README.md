# UnMute
UnMute-time feedback and is an AI‑powered learning assistant that helps users improve their English pronunciation by providing immediate feedback and actionable tips.

## 🚀 Getting Started
Users can access the main site and log in using their Google or Facebook account. Learning begins with the first call.

## 🧩 Features
One-on-one live audio sessions with an agent, designed to help users develop a British English accent.

## 🤖 AI model
It is configurable with NEXT_PUBLIC_GEMINI_MODEL environment variable. Currently it is
Google Gemini - **gemini-2.5-flash-native-audio-latest**
